Question-1: What are the different ways to select an element in the DOM?

There are many different ways to select element in DOM for example getElementById (it's select by id), getElementByClassName (its select by class name), getElementByTagName (its select by tag name), querySelector (its select CSS single element by CSS selector), querySelectorAll (its select multiple CSS element by CSS selector).

Question-2: What is the difference between innerHTML, innerText, and textContent?

In JavaScript, "innerHTML" retrieves the complete HTML content within an element, including tags, while "innerText" returns only the visible text content of an element (ignoring styling), and "textContent" retrieves all text content within an element, including hidden elements and whitespace, but without interpreting any HTML tags; essentially, "innerText" is more visually-oriented, while "textContent" provides raw text data.

Question-3: What is event delegation in the DOM?

In the DOM (Document Object Model), "event delegation" is a technique where you attach a single event listener to a parent element, allowing it to handle events triggered on its child elements, instead of adding separate listeners to each child element, effectively "delegating" the event handling to the parent; this leverages the event bubbling mechanism where events propagate upwards through the DOM tree, making it more efficient for large lists of dynamic elements.

Question-4: What is event bubbling in the DOM?

In the DOM (Document Object Model), "event bubbling" refers to the process where when an event occurs on an element, it triggers not only on that element but also on its parent elements in the DOM hierarchy, essentially "bubbling up" the event until it reaches the topmost parent element, like the document itself; allowing you to potentially handle the event at any level of the hierarchy with a single event listener on a parent element.

Question-5: How do you create, add, and remove elements using JavaScript?

An index is the position of an element in the array. The first element in the array has an index of 0, the second element has an index of 1, and so on. To add elements at a specific index, you can use the splice method. The splice method allows you to add and remove elements from a specific index in an array.
